# Ogre

Simplified health monitoring and telemetry for Docker containers in modern day
distributed systems.

## Getting Started

Ensure you have Go installed. Running Ogre then should be as simple as:
```console
$ make build
$ ./bin/ogre
```

### Testing

``make test``
